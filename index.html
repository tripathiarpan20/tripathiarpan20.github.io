Current `Projects

Facial Keypoint mapping using MobileNetv2 feature extractor

Even though MobileNetv2 was trained on ImageNet dataset, the convolutional filters of the pretrained network can be used to extract representational features, even for faces, given the availability of enough training/val data to train a regressor for mapping the activations of final convolutional filters onto keypoint features of faces.

The model is comparatively small in size (~15 MB, as compared to ~500 MB sized VGG-16 network) but performs really well on validation set. Pairing this model with haarcascade_face in OpenCV to first detect faces in order to mark keypoint features works well with limited number of faces

    • Neural Style Transfer implementation in Keras

    In a pretrained network, the activations of layers capture properties of the image, with increasing complexity in deeper layers, such activation pattern capture the 'content' of the image and the covariance matrix of particular layers by two images capture similarity in 'style' of these images, optimization of one image with respect to the weighted sum of these two objectives can project the style of one image onto the other

  • Notebook on Github

    • Converting random noise into an image using a pretrained Neural Network

    Most available pretrained neural networks were trained on 1000 classes of ImageNet dataset, the pretrained weights can be used to convert gaussian noise into one of the 1000 images classes by maximizing first Classifier layer with respect to target output neuron(Gradient Ascent), then maximizing the input image with respect to the first Classifier layer (Feature Inversion)
